# Documentation Retrieval RAG Chatbot

## Project Overview
The Documentation Retrieval RAG Chatbot is a Generative AI-based application that retrieves relevant information from documents and generates intelligent responses using Large Language Models (LLMs).

This project uses Retrieval-Augmented Generation (RAG) architecture to improve response accuracy by combining semantic search with LLM-generated answers.

---

## Features
- Document Upload
- Text Chunking
- Embedding Generation
- Vector Database Storage
- Semantic Search
- Relevant Chunk Retrieval
- LLM Response Generation
- Interactive Chat Interface

## RAG Workflow

1. Document Upload
2. Text Extraction
3. Text Chunking
4. Embedding Generation
5. Store Embeddings in Vector Database
6. Semantic Search
7. Retrieve Relevant Chunks
8. Generate Response using LLM
